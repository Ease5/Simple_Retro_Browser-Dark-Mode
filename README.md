# Simple Retro Browser - Dark Mode

A lightweight Python-based web browser with a retro dark mode theme.

## Features
- Simple and easy-to-use interface
- Dark mode for comfortable night-time browsing
- Minimal dependencies

## How to run
1. Make sure you have Python installed
2. Run the script: `browser.py`

# Disclaimers
1. This a basic browser and doesn't feature much security
2. The browser doesnt save cookies when the session end so you will have to sign in to your website everytime you open the browser
3. There is no history or settings menu
4. Websites will occasionally ask for security verification like CAPTCHAS etc.

# Features
1. Back button
2. Forward button
3. Home button
4. Tabs
5. Draggable adress bar
6. Search engine: Google
